#Design Pattern Facade

Projeto criado para fins de aprendizado dos principios do Padrão de Projeto Façade

Facade -> 
	* é um padrão que cria uma interface simples de alto nível para interfaces complexas, possibilitando que um modulo, serviço ou framework apartado dependa apenas da interface de maior nível.

Quando usar ->
	* você tem uma biblioteca definida e apartada da aplicação client e não quer que a mesma acesse recursos específicos dentro dela;
	* você precisa simplificar uma sequência de passos mais complexa;
	* você quer ter diferentes interfaces com o usuário;
	* você precisa de um novo ponto de acesso a um recurso de um software em camadas.
